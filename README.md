# Hubo Configurator - Transparante Tafelbeschermers

**Bedrijf:** Finesse Decpartners  
**Contact:** Vincent  
**Voor:** Hubo.be integratie

## Wat is dit?

Dit is een configurator voor transparante tafelbeschermers die in een iframe op de Hubo website draait. Klanten kunnen hun gewenste vorm en afmetingen kiezen, en het product wordt direct aan hun winkelmandje toegevoegd.

## Bestanden

- `index.html` - Nederlandse versie van de configurator
- `index-fr.html` - Franse versie van de configurator (TODO)
- `parent-test.html` - Testpagina voor lokale ontwikkeling
- `PLA2100.jpg` - Product afbeelding
- `rounded-corner-expl.png` - Uitleg afgeronde hoeken
